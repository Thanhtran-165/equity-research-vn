      <div class="gloss-item"><div class="gloss-term">Backlog</div><div class="gloss-def">Tổng giá trị hợp đồng đã ký nhưng chưa ghi nhận doanh thu. Đảm bảo doanh thu tương lai.</div></div>

      <div class="gloss-item"><div class="gloss-term">ROE</div><div class="gloss-def">Return on Equity: LNST / Vốn chủ sở hữu. Đo hiệu quả sử dụng vốn cổ đông.</div></div>
      <div class="gloss-item"><div class="gloss-term">P/E</div><div class="gloss-def">Price-to-Earnings: Giá / EPS. Số năm thu hồi vốn qua lợi nhuận (nếu không tăng trưởng).</div></div>
      <div class="gloss-item"><div class="gloss-term">P/B</div><div class="gloss-def">Price-to-Book: Giá / BVPS. So sánh giá cổ phiếu với giá trị sổ sách tài sản.</div></div>

      <div class="gloss-item"><div class="gloss-term">CAGR</div><div class="gloss-def">Compound Annual Growth Rate: Tốc độ tăng trưởng kép hàng năm trong một giai đoạn.</div></div>
      <div class="gloss-item"><div class="gloss-term">Split-adjusted</div><div class="gloss-def">Dữ liệu giá/EPS đã điều chỉnh cho các đợt phát hành cổ phiếu mới, chia tách. Giúp so sánh liên tục.</div></div>
      <div class="gloss-item"><div class="gloss-term">Cross-check</div><div class="gloss-def">Kiểm tra chéo: đối chiếu số liệu từ 2+ nguồn độc lập để xác nhận độ tin cậy.</div></div>

    <li><strong>Nếu là tiền của tôi.</strong> Trước khi quyết định, hỏi: "Tôi có dám đặt $800 triệu dựa trên con số này không?" Nếu chưa dám → verify thêm.</li>
  </ol>
  <div class="disclaimer-title sub">Nhãn chất lượng dữ liệu</div>
  <p>Mỗi số có nhãn ở bảng Data Quality bên dưới:</p>

function grad(ctx, c1, c2){
  const chart = ctx.chart;
  const {ctx: cc, chartArea} = chart;
  if (!chartArea) return c1;
  const g = cc.createLinearGradient(0, chartArea.top, 0, chartArea.bottom);
  g.addColorStop(0, c1); g.addColorStop(1, c2);
  return g;
}
