<section id="sec-glossary" class="section">
  <div class="section-title aos"><span class="num">21</span><h2>📖 Hướng dẫn đọc — Giải thích thuật ngữ</h2><div class="divider"></div></div>
  <div class="section-sub">Thuật ngữ tài chính dùng trong báo cáo.</div>
  <div class="glossary-grid aos">
    <div class="gloss-item"><div class="gloss-term">P/E (Price/Earnings)</div><div class="gloss-def">Giá / EPS. Định giá cổ phiếu bao nhiêu năm thu hồi vốn. P/E thấp = rẻ (nhưng có thể có lý do).</div></div>
    <div class="gloss-item"><div class="gloss-term">P/B (Price/Book)</div><div class="gloss-def">Giá / BVPS. So sánh giá với giá trị sổ sách. P/B < 1 = rẻ hơn vốn chủ sở hữu.</div></div>
    <div class="gloss-item"><div class="gloss-term">EPS</div><div class="gloss-def">Lợi nhuận trên mỗi cổ phiếu (VND/cp).</div></div>
    <div class="gloss-item"><div class="gloss-term">ROE</div><div class="gloss-def">Lợi nhuận / Vốn CSH. Hiệu quả sử dụng vốn (%)</div></div>
    <div class="gloss-item"><div class="gloss-term">CFO</div><div class="gloss-def">Dòng tiền hoạt động kinh doanh. CFO âm = tiền không về (dù có lợi nhuận).</div></div>
    <div class="gloss-item"><div class="gloss-term">Backlog</div><div class="gloss-def">Tổng giá trị hợp đồng chưa hoàn thành. Backlog lớn = doanh thu tương lai đảm bảo.</div></div>
    <div class="gloss-item"><div class="gloss-term">MA (Moving Average)</div><div class="gloss-def">Trung bình động. MA50 = trung bình 50 phiên. Giá trên MA = trend tăng.</div></div>
    <div class="gloss-item"><div class="gloss-term">RSI</div><div class="gloss-def">Chỉ số sức mạnh tương đối. RSI < 30 = oversold, > 70 = overbought.</div></div>
    <div class="gloss-item"><div class="gloss-term">Drawdown</div><div class="gloss-def">Khoảng giảm từ đỉnh giá nhất. Drawdown -50% = mất nửa giá trị.</div></div>
    <div class="gloss-item"><div class="gloss-term">Split-adjusted</div><div class="gloss-def">Dữ liệu đã điều chỉnh chia tách cổ phiếu. So sánh chính xác qua thời gian.</div></div>
    <div class="gloss-item"><div class="gloss-term">Bẫy 5B</div><div class="gloss-def">Báo cáo đẹp nhưng tiền không về. Lợi nhuận cao nhưng CFO âm.</div></div>
    <div class="gloss-item"><div class="gloss-term">Cross-check</div><div class="gloss-def">Đối chiếu nhiều nguồn độc lập để xác minh số liệu.</div></div>
  </div>
</section>

(function(){
  const nav = document.getElementById('topnav');
  const links = Array.from(document.querySelectorAll('.topnav-link'));
  const progressBar = document.getElementById('progressBar');
  const backTop = document.getElementById('backTop');
  const sections = links.map(l => document.getElementById(l.dataset.target)).filter(Boolean);

  function onScroll(){
    const y = window.scrollY;
    const h = document.documentElement.scrollHeight - window.innerHeight;
    progressBar.style.width = (h>0 ? (y/h*100) : 0) + '%';
    if (y > 600) backTop.classList.add('visible'); else backTop.classList.remove('visible');
    const navH = nav.offsetHeight + 10;
    let active = sections[0];
    for (const s of sections){
      if (s && (s.getBoundingClientRect().top - navH) <= 0) active = s;
    }
    links.forEach(l => {
      l.classList.toggle('active', active && l.dataset.target === active.id);
    });
    const activeLink = links.find(l => l.classList.contains('active'));
    if (activeLink){
      const navInner = nav.querySelector('.topnav-inner');
      const linkRect = activeLink.getBoundingClientRect();
      const innerRect = navInner.getBoundingClientRect();
      if (linkRect.right > innerRect.right - 20 || linkRect.left < innerRect.left + 20){
        navInner.scrollTo({ left: navInner.scrollLeft + (linkRect.left - innerRect.left - 40), behavior:'smooth' });
      }
    }
  }
  window.addEventListener('scroll', onScroll, {passive:true});
  window.addEventListener('resize', onScroll);
  links.forEach(l => {
    l.addEventListener('click', (e)=>{
      const t = document.getElementById(l.dataset.target);
      if (!t) return;
      e.preventDefault();
      const top = t.getBoundingClientRect().top + window.scrollY - nav.offsetHeight - 8;
      window.scrollTo({ top, behavior:'smooth' });
    });
  });
  backTop.addEventListener('click', ()=> window.scrollTo({top:0, behavior:'smooth'}));
  onScroll();
})();
</script>
