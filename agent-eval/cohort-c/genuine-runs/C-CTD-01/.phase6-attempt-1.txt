  <div class="section-title aos"><span class="num" style="background:linear-gradient(135deg,#f5a623,#4a9eff);color:#0f1419">★</span><h2>Special Insight 1 — Sự mâu thuẫn giữa Fundamental và Technical</h2><div class="divider"></div></div>
  <div class="insight-frame aos">
    <p style="font-size:14px;color:var(--text);line-height:1.7">CTD hiện tại là một case study kinh điển về <strong>mâu thuẫn giữa tín hiệu fundamental và technical</strong>. Về cơ bản, công ty đang có giai đoạn tốt nhất trong 5 năm qua: doanh thu 28,561 tỷ VND, lợi nhuận 1,510 tỷ VND, EPS tăng từ 280 VND (2021) lên 18,357 VND (2025) — tăng 65 lần. P/E chỉ 3.91×, P/B 1.0×.</p>
    <p style="font-size:14px;color:var(--text);line-height:1.7;margin-top:12px">Tuy nhiên, tech score lại ở mức <strong>-6/6 (STRONG SELL)</strong> — mức cực đoan nhất. Điều này có nghĩa là: (1) thị trường không tin lợi nhuận hiện tại là bền vững, hoặc (2) có rủi ro tiềm ẩn chưa được phản ánh trong BCTC, hoặc (3) dòng tiền đang rút khỏi cổ phiếu này vì lý do thanh khoản.</p>
    <div class="callout plain" style="margin-top:16px">
      <div class="callout-title">💡 Nói cách khác</div>
      <div class="callout-body">Đây là dạng "bẫy giá trị" (value trap) cổ điển. Cổ phiếu rẻ vì lý do — thị trường thường biết điều mà BCTC chưa cho thấy. <strong>Quy tắc: không bắt dao rơi</strong>. Chờ đợi tech score cải thiện trước khi xem xét.</div>
    </div>
    <div class="callout honest-correction" style="margin-top:16px">
      <div class="callout-title">⚠️ Honest Limitation</div>
      <div class="callout-body">Chúng tôi không có dữ liệu chi tiết về lý do tech score -6/6. Có thể do xu hướng giá, thanh khoản, hoặc tin tức tiêu cực. Cần <strong>cross-check với tin tức gần đây</strong> để hiểu nguyên nhân.</div>

<section id="sec-insight-2" class="section">
  <div class="section-title aos"><span class="num" style="background:linear-gradient(135deg,#f5a623,#4a9eff);color:#0f1419">★</span><h2>Special Insight 2 — Bí ẩn Vốn chủ sở hữu giảm</h2><div class="divider"></div></div>
  <div class="insight-frame aos">
    <p style="font-size:14px;color:var(--text);line-height:1.7">Một điểm bất thường trong BCTC của CTD: <strong>Vốn chủ sở hữu giảm từ 8,470 tỷ (2024) xuống 7,962 tỷ (2025)</strong>, giảm 508 tỷ VND. Điều này xảy ra <em>dù</em> lợi nhuận sau thuế tăng từ 711 tỷ lên 1,510 tỷ (tăng 799 tỷ).</p>
    <p style="font-size:14px;color:var(--text);line-height:1.7;margin-top:12px">Về mặt kế toán, VCSH = VCSH đầu kỳ + LNST - Cổ tức - Cổ phiếu quỹ + Other comprehensive income. Nếu LNST tăng 799 tỷ mà VCSH giảm 508 tỷ, thì <strong>cổ tức hoặc cổ phiếu quỹ phải rất lớn</strong> (khoảng 1,300+ tỷ VND). Đây là tín hiệu tích cực nếu công ty trả cổ tức tiền mặt lớn cho cổ đông.</p>
    <div class="callout plain" style="margin-top:16px">

      <div class="callout-body">Nếu CTD chia cổ tức 1,300 tỷ VND cho 112 triệu cổ phiếu, tức là ~11,600 VND/cổ phiếu — <strong>dividend yield ~16%</strong> ở giá 71,700 VND. Đây là mức cực kỳ cao. Tuy nhiên, cần xác nhận từ BCTC chi tiết.</div>

  <div class="insight-frame aos">
    <p style="font-size:14px;color:var(--text);line-height:1.7">CTD thể hiện <strong>đòn bẩy hoạt động (operating leverage) rất mạnh</strong>. Khi doanh thu tăng 20% (từ 23,733 tỷ lên 28,561 tỷ), lợi nhuận lại tăng 112% (từ 711 tỷ lên 1,510 tỷ). Tỷ lệ đòn bẩy ~5.6× — mỗi 1% tăng doanh thu tạo ra 5.6% tăng lợi nhuận.</p>
    <p style="font-size:14px;color:var(--text);line-height:1.7;margin-top:12px">Biên lợi nhuận ròng (net margin) cải thiện từ 3.0% (2024) lên 5.3% (2025). Nếu xu hướng này tiếp tục và biên LN đạt 6-7% (mức trung bình ngành), lợi nhuận 2026 có thể đạt 2,000-2,500 tỷ VND.</p>
    <div class="callout plain" style="margin-top:16px">
      <div class="callout-title">💡 Nói cách khác</div>
      <div class="callout-body">CTD giống một "con đà" — khi công việc đầy đủ, chi phí cố định được trải đều, biên LN nở rộng rất nhanh. Nhưng <strong>ngược lại</strong>, nếu doanh thu giảm, LN cũng thu hẹp nhanh chóng. Đây là đặc tính của ngành xây dựng.</div>
    </div>
    <div class="callout info" style="margin-top:16px">
      <div class="callout-title">Tóm lại</div>
      <div class="callout-body">Đòn bẩy hoạt động mạnh là con dao hai lưỡi. Trong giai đoạn phục hồi (2023-2025), nó tạo ra siêu tăng trưởng. Nhưng nếu chu kỳ quay xuống, LN có thể sụt giảm nhanh hơn doanh thu.</div>
    </div>

function grad(ctx, c1, c2){
  const chart = ctx.chart;
  const {ctx: cc, chartArea} = chart;
  if (!chartArea) return c1;
  const g = cc.createLinearGradient(0, chartArea.top, 0, chartArea.bottom);
  g.addColorStop(0, c1); g.addColorStop(1, c2);
  return g;
}

(function(){
  const nav = document.getElementById('topnav');
  const links = Array.from(document.querySelectorAll('.topnav-link'));
  const progressBar = document.getElementById('progressBar');
  const backTop = document.getElementById('backTop');
  const sections = links.map(l => document.getElementById(l.dataset.target)).filter(Boolean);

  function onScroll(){
    const y = window.scrollY;
    const h = document.documentElement.scrollHeight - window.innerHeight;
    progressBar.style.width = (h>0 ? (y/h*100) : 0) + '%';
    if (y > 600) backTop.classList.add('visible'); else backTop.classList.remove('visible');
    const navH = nav.offsetHeight + 10;
    let active = sections[0];
    for (const s of sections){
      if (s && (s.getBoundingClientRect().top - navH) <= 0) active = s;
    }
    links.forEach(l => {
      l.classList.toggle('active', active && l.dataset.target === active.id);
    });
    const activeLink = links.find(l => l.classList.contains('active'));
    if (activeLink){
      const navInner = nav.querySelector('.topnav-inner');
      const linkRect = activeLink.getBoundingClientRect();
      const innerRect = navInner.getBoundingClientRect();
      if (linkRect.right > innerRect.right - 20 || linkRect.left < innerRect.left + 20){
        navInner.scrollTo({ left: navInner.scrollLeft + (linkRect.left - innerRect.left - 40), behavior:'smooth' });
      }
    }
  }
  window.addEventListener('scroll', onScroll, {passive:true});
  window.addEventListener('resize', onScroll);
  links.forEach(l => {
    l.addEventListener('click', (e)=>{
      const t = document.getElementById(l.dataset.target);
      if (!t) return;
      e.preventDefault();
      const top = t.getBoundingClientRect().top + window.scrollY - nav.offsetHeight - 8;
      window.scrollTo({ top, behavior:'smooth' });
    });
  });
  backTop.addEventListener('click', ()=> window.scrollTo({top:0, behavior:'smooth'}));
  onScroll();
})();
