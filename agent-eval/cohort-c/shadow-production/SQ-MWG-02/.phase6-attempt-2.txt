Lý do: MWG <strong>chia cổ tức đặc biệt</strong> (bằng tiền mặt và cổ phiếu) cho cổ đông, rút VCSH ra ngoài.<a class="ref" href="#ref-7">7</a></div></div>

Khi VCSH giảm mạnh, BVPS giảm → P/B tăng cơ học. <strong>P/B 12.9x không phản ánh "đắt" mà phản ánh "VCSH mỏng".</strong> Nếu điều chỉnh lại VCSH về mức 2021, P/B thực chỉ ~4-5x.</div></div>

Nếu ai đó nhìn vào và nói "tài khoản này nhỏ quá, bạn nghèo" — họ sai. Bạn chỉ đã <strong>rút tiền ra</strong>. MWG cũng vậy: họ <strong>trả tiền cho cổ đông</strong> thay vì giữ lại VCSH. P/B 12.9x là <strong>hiệu ứng split-adjusted ngược</strong> — cần hiểu bản chất trước khi kết luận.</div>

      <div class="callout-body">Phân tích này là <strong>ước tính</strong> và có <strong>limitation</strong>. Không có số liệu chính thức về lượng cổ tức đặc biệt MWG đã trả trong 2021-2025 trong gói dữ liệu này. Bạn nên cross-check với BCTC kiểm toán để xác nhận.</div>

<section id="sec-glossary" class="section">
  <div class="section-title aos"><span class="num">21</span><h2>📖 Hướng dẫn đọc — Giải thích thuật ngữ</h2><div class="divider"></div></div>
  <div class="section-sub">Các thuật ngữ tài chính dùng trong báo cáo.</div>
  <div class="glossary-grid aos">
    <div class="gloss-item"><div class="gloss-term">P/E (Price/Earnings)</div><div class="gloss-def">Giá / EPS. Cho biết bạn trả bao nhiêu cho 1 đồng lợi nhuận. P/E 11.7x = trả 11.7 đồng cho 1 đồng LN.</div></div>
    <div class="gloss-item"><div class="gloss-term">P/B (Price/Book)</div><div class="gloss-def">Giá / BVPS. So sánh giá cổ phiếu với giá trị sổ sách. P/B cao có thể do VCSH mỏng (như MWG).</div></div>
    <div class="gloss-item"><div class="gloss-term">EPS</div><div class="gloss-def">Lợi nhuận trên mỗi cổ phiếu. MWG EPS 2025 = 6,689 VND/cổ phiếu.</div></div>
    <div class="gloss-item"><div class="gloss-term">BVPS</div><div class="gloss-def">Giá trị sổ sách trên mỗi cổ phiếu = VCSH / số lượng cổ phiếu.</div></div>
    <div class="gloss-item"><div class="gloss-term">Capex</div><div class="gloss-def">Chi phí vốn — tiền đầu tư vào tài sản cố định (mở cửa hàng, mua thiết bị).</div></div>
    <div class="gloss-item"><div class="gloss-term">FCF (Free Cash Flow)</div><div class="gloss-def">Dòng tiền tự do = CFO - Capex. Tiền thừa sau khi đầu tư.</div></div>
    <div class="gloss-item"><div class="gloss-term">Split-adjusted</div><div class="gloss-def">Điều chỉnh do chia tách cổ phiếu. MWG chia cổ tức bằng cổ phiếu → EPS/VCSH cần adjust để so sánh.</div></div>
    <div class="gloss-item"><div class="gloss-term">Turnaround</div><div class="gloss-def">Phục hồi — công ty từ lỗ/quán tính xấu chuyển sang sinh lời/tăng trưởng.</div></div>
    <div class="gloss-item"><div class="gloss-term">Cross-check</div><div class="gloss-def">Đối chiếu chéo — kiểm tra số liệu từ 2+ nguồn độc lập để xác nhận.</div></div>
    <div class="gloss-item"><div class="gloss-term">VCSH</div><div class="gloss-def">Vốn chủ sở hữu = Tổng tài sản - Nợ phải trả. Tiền của cổ đông trong công ty.</div></div>
  </div>
</section>

(function(){
  const nav = document.getElementById('topnav');
  const links = Array.from(document.querySelectorAll('.topnav-link'));
  const progressBar = document.getElementById('progressBar');
  const backTop = document.getElementById('backTop');
  const sections = links.map(l => document.getElementById(l.dataset.target)).filter(Boolean);

  function onScroll(){
    const y = window.scrollY;
    const h = document.documentElement.scrollHeight - window.innerHeight;
    progressBar.style.width = (h>0 ? (y/h*100) : 0) + '%';
    if (y > 600) backTop.classList.add('visible'); else backTop.classList.remove('visible');
    const navH = nav.offsetHeight + 10;
    let active = sections[0];
    for (const s of sections){
      if (s && (s.getBoundingClientRect().top - navH) <= 0) active = s;
    }
    links.forEach(l => {
      l.classList.toggle('active', active && l.dataset.target === active.id);
    });
    const activeLink = links.find(l => l.classList.contains('active'));
    if (activeLink){
      const navInner = nav.querySelector('.topnav-inner');
      const linkRect = activeLink.getBoundingClientRect();
      const innerRect = navInner.getBoundingClientRect();
      if (linkRect.right > innerRect.right - 20 || linkRect.left < innerRect.left + 20){
        navInner.scrollTo({ left: navInner.scrollLeft + (linkRect.left - innerRect.left - 40), behavior:'smooth' });
      }
    }
  }
  window.addEventListener('scroll', onScroll, {passive:true});
  window.addEventListener('resize', onScroll);
  links.forEach(l => {
    l.addEventListener('click', (e)=>{
      const t = document.getElementById(l.dataset.target);
      if (!t) return;
      e.preventDefault();
      const top = t.getBoundingClientRect().top + window.scrollY - nav.offsetHeight - 8;
      window.scrollTo({ top, behavior:'smooth' });
    });
  });
  backTop.addEventListener('click', ()=> window.scrollTo({top:0, behavior:'smooth'}));
  onScroll();
})();
</script>
