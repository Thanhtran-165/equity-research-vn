<section id="sec-glossary" class="section">
  <div class="section-title aos"><span class="num">21</span><h2>📖 Hướng dẫn đọc — Giải thích thuật ngữ</h2><div class="divider"></div></div>
  <div class="section-sub">Các thuật ngữ tài chính sử dụng trong báo cáo.</div>
  <div class="glossary-grid aos">
    <div class="gloss-item"><div class="gloss-term">P/E (Price/Earnings)</div><div class="gloss-def">Giá / EPS. Đo lường giá trả cho 1 đồng lợi nhuận. P/E thấp = định giá rẻ (nhưng có thể vì lý do chính đáng).</div></div>
    <div class="gloss-item"><div class="gloss-term">P/B (Price/Book)</div><div class="gloss-def">Giá / BVPS. Đo lường giá so với giá trị sổ sách. P/B cao có thể do trả cổ tức lớn (như SAB).</div></div>
    <div class="gloss-item"><div class="gloss-term">ROE (Return on Equity)</div><div class="gloss-def">Lợi nhuận / Vốn chủ sở hữu. Đo lường hiệu quả sử dụng vốn. ROE >20% = rất tốt.</div></div>
    <div class="gloss-item"><div class="gloss-term">EPS</div><div class="gloss-def">Lợi nhuận trên mỗi cổ phiếu. = LNST / Số cổ phiếu. Đơn vị: VND.</div></div>
    <div class="gloss-item"><div class="gloss-term">CFO (Cash from Operations)</div><div class="gloss-def">Dòng tiền từ hoạt động kinh doanh. Dương và tăng = doanh nghiệp khỏe mạnh.</div></div>
    <div class="gloss-item"><div class="gloss-term">Capex</div><div class="gloss-def">Chi phí vốn (mua sắm tài sản cố định). Capex thấp = ít cần đầu tư mới.</div></div>
    <div class="gloss-item"><div class="gloss-term">Split-adjusted</div><div class="gloss-def">Điều chỉnh cho các sự kiện chia cổ phiếu. Đảm bảo so sánh lịch sử chính xác.</div></div>
    <div class="gloss-item"><div class="gloss-term">TTBĐ</div><div class="gloss-def">Thuế tiêu thụ đặc biệt. Áp dụng cho bia, rượu, thuốc lá. Rủi ro chính sách lớn cho SAB.</div></div>
    <div class="gloss-item"><div class="gloss-term">DCA</div><div class="gloss-def">Dollar Cost Averaging — chia nhỏ số tiền đầu tư thành nhiều đợt để giảm rủi ro thời điểm.</div></div>
    <div class="gloss-item"><div class="gloss-term">Bẫy 5F</div><div class="gloss-def">Cổ phiếu rẻ nhưng tiếp tục giảm — "đắt cho một lý do". Cẩn thận với P/E thấp khi doanh thu giảm.</div></div>

(function(){
  const nav = document.getElementById('topnav');
  const links = Array.from(document.querySelectorAll('.topnav-link'));
  const progressBar = document.getElementById('progressBar');
  const backTop = document.getElementById('backTop');
  const sections = links.map(l => document.getElementById(l.dataset.target)).filter(Boolean);

  function onScroll(){
    const y = window.scrollY;
    const h = document.documentElement.scrollHeight - window.innerHeight;
    progressBar.style.width = (h>0 ? (y/h*100) : 0) + '%';
    if (y > 600) backTop.classList.add('visible'); else backTop.classList.remove('visible');
    const navH = nav.offsetHeight + 10;
    let active = sections[0];
    for (const s of sections){
      if (s && (s.getBoundingClientRect().top - navH) <= 0) active = s;
    }
    links.forEach(l => {
      l.classList.toggle('active', active && l.dataset.target === active.id);
    });
    const activeLink = links.find(l => l.classList.contains('active'));
    if (activeLink){
      const navInner = nav.querySelector('.topnav-inner');
      const linkRect = activeLink.getBoundingClientRect();
      const innerRect = navInner.getBoundingClientRect();
      if (linkRect.right > innerRect.right - 20 || linkRect.left < innerRect.left + 20){
        navInner.scrollTo({ left: navInner.scrollLeft + (linkRect.left - innerRect.left - 40), behavior:'smooth' });
      }
    }
  }
  window.addEventListener('scroll', onScroll, {passive:true});
  window.addEventListener('resize', onScroll);
  links.forEach(l => {
    l.addEventListener('click', (e)=>{
      const t = document.getElementById(l.dataset.target);
      if (!t) return;
      e.preventDefault();
      const top = t.getBoundingClientRect().top + window.scrollY - nav.offsetHeight - 8;
      window.scrollTo({ top, behavior:'smooth' });
    });
  });
  backTop.addEventListener('click', ()=> window.scrollTo({top:0, behavior:'smooth'}));
  onScroll();
})();
</script>
