<section id="sec-glossary" class="section">
  <div class="section-title aos"><span class="num">21</span><h2>📖 Hướng dẫn đọc — Giải thích thuật ngữ</h2><div class="divider"></div></div>
  <div class="section-sub">Thuật ngữ tài chính · Cách đọc báo cáo</div>
  <div class="glossary-grid aos">
    <div class="gloss-item"><div class="gloss-term">P/E (Price/Earnings)</div><div class="gloss-def">Giá cổ phiếu chia cho EPS. P/E 10× nghĩa là nhà đầu tư sẵn sàng trả 10 lần lợi nhuận 1 năm. Thấp hơn = rẻ hơn.</div></div>
    <div class="gloss-item"><div class="gloss-term">P/B (Price/Book)</div><div class="gloss-def">Giá cổ phiếu chia cho giá trị sổ sách (BVPS). P/B 4.8× nghĩa là giá gấp 4.8 lần vốn chủ sở hữu trên cổ phiếu.</div></div>
    <div class="gloss-item"><div class="gloss-term">EPS (Earnings Per Share)</div><div class="gloss-def">Lợi nhuận trên mỗi cổ phiếu. SSI 2025: 2,428 VND/cổ phiếu.</div></div>
    <div class="gloss-item"><div class="gloss-term">ROE (Return on Equity)</div><div class="gloss-def">Lợi nhuận chia vốn chủ sở hữu. ROE cao = hiệu quả sử dụng vốn tốt.</div></div>
    <div class="gloss-item"><div class="gloss-term">Margin (Cho vay ký quỹ)</div><div class="gloss-def">CTCK cho khách vay tiền mua cổ phiếu, thu lãi. Nguồn doanh thu lớn nhưng rủi ro nợ xấu.</div></div>
    <div class="gloss-item"><div class="gloss-term">Tự doanh</div><div class="gloss-def">CTCK dùng vốn tự có đầu tư chứng khoán. Lợi nhuận đòn bẩy nhưng rủi ro thị trường.</div></div>
    <div class="gloss-item"><div class="gloss-term">Split-adjusted</div><div class="gloss-def">Dữ liệu được điều chỉnh theo chia tách cổ phiếu. Cần kiểm tra khi so sánh lịch sử.</div></div>
    <div class="gloss-item"><div class="gloss-term">Bẫy 5B</div><div class="gloss-def">Bị mắc kẹt trong vùng giá trị nhưng giá không tăng — cổ phiếu rẻ nhưng không tăng.</div></div>
    <div class="gloss-item"><div class="gloss-term">Cross-check</div><div class="gloss-def">Đối chiếu nhiều nguồn để xác minh dữ liệu. Kỷ luật bắt buộc khi đọc báo cáo AI.</div></div>

(function(){
  const nav = document.getElementById('topnav');
  const links = Array.from(document.querySelectorAll('.topnav-link'));
  const progressBar = document.getElementById('progressBar');
  const backTop = document.getElementById('backTop');
  const sections = links.map(l => document.getElementById(l.dataset.target)).filter(Boolean);

  function onScroll(){
    const y = window.scrollY;
    const h = document.documentElement.scrollHeight - window.innerHeight;
    progressBar.style.width = (h>0 ? (y/h*100) : 0) + '%';
    if (y > 600) backTop.classList.add('visible'); else backTop.classList.remove('visible');
    const navH = nav.offsetHeight + 10;
    let active = sections[0];
    for (const s of sections){
      if (s && (s.getBoundingClientRect().top - navH) <= 0) active = s;
    }
    links.forEach(l => {
      l.classList.toggle('active', active && l.dataset.target === active.id);
    });
    const activeLink = links.find(l => l.classList.contains('active'));
    if (activeLink){
      const navInner = nav.querySelector('.topnav-inner');
      const linkRect = activeLink.getBoundingClientRect();
      const innerRect = navInner.getBoundingClientRect();
      if (linkRect.right > innerRect.right - 20 || linkRect.left < innerRect.left + 20){
        navInner.scrollTo({ left: navInner.scrollLeft + (linkRect.left - innerRect.left - 40), behavior:'smooth' });
      }
    }
  }
  window.addEventListener('scroll', onScroll, {passive:true});
  window.addEventListener('resize', onScroll);
  links.forEach(l => {
    l.addEventListener('click', (e)=>{
      const t = document.getElementById(l.dataset.target);
      if (!t) return;
      e.preventDefault();
      const top = t.getBoundingClientRect().top + window.scrollY - nav.offsetHeight - 8;
      window.scrollTo({ top, behavior:'smooth' });
    });
  });
  backTop.addEventListener('click', ()=> window.scrollTo({top:0, behavior:'smooth'}));
  onScroll();
})();

</script>
