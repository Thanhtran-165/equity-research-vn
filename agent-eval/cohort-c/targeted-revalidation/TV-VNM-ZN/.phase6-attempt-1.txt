<section id="sec-glossary" class="section">
  <div class="section-title aos"><span class="num">21</span><h2>📖 Hướng dẫn đọc — Giải thích thuật ngữ</h2><div class="divider"></div></div>
  <div class="section-sub">Các thuật ngữ tài chính sử dụng trong báo cáo.</div>
  <div class="aos">
    <div class="glossary-grid">
      <div class="gloss-item"><div class="gloss-term">P/E (Price/Earnings)</div><div class="gloss-def">Giá cổ phiếu chia cho EPS. Cho biết bạn trả bao nhiêu cho 1 đồng lợi nhuận. P/E thấp = rẻ hơn (nhưng có thể rẻ vì lý do).</div></div>
      <div class="gloss-item"><div class="gloss-term">P/B (Price/Book)</div><div class="gloss-def">Giá cổ phiếu chia cho giá trị sổ sách. Cho biết bạn trả bao nhiêu cho 1 đồng tài sản ròng.</div></div>
      <div class="gloss-item"><div class="gloss-term">ROE (Return on Equity)</div><div class="gloss-def">Lợi nhuận ròng chia vốn chủ sở hữu. Đo lường hiệu quả sử dụng vốn của cổ đông. >20% = rất tốt.</div></div>
      <div class="gloss-item"><div class="gloss-term">EPS (Earnings Per Share)</div><div class="gloss-def">Lợi nhuận trên mỗi cổ phiếu. VNM EPS = 5,295₫ nghĩa là mỗi cổ phiếu tạo ra 5,295₫ lợi nhuận/năm.</div></div>
      <div class="gloss-item"><div class="gloss-term">Capex (Capital Expenditure)</div><div class="gloss-def">Chi phí đầu tư tài sản cố định (nhà máy, trang trại, thiết bị). Capex cao = đang đầu tư cho tương lai.</div></div>
      <div class="gloss-item"><div class="gloss-term">FCF (Free Cash Flow)</div><div class="gloss-def">Dòng tiền tự do = CFO - Capex. Tiền thừa sau khi đầu tư, dùng để trả cổ tức hoặc mua lại cổ phiếu.</div></div>
      <div class="gloss-item"><div class="gloss-term">Beta</div><div class="gloss-def">Độ nhạy cảm giá cổ phiếu so với thị trường. Beta <1 = ít biến động hơn thị trường (phòng thủ).</div></div>
      <div class="gloss-item"><div class="gloss-term">Drawdown</div><div class="gloss-def">Phần trăm sụt giảm từ đỉnh giá. Drawdown -30% = giá giảm 30% so với đỉnh.</div></div>
    </div>
  </div>

  const links = Array.from(document.querySelectorAll('.topnav-link'));
  const progressBar = document.getElementById('progressBar');
  const backTop = document.getElementById('backTop');
  const sections = links.map(l => document.getElementById(l.dataset.target)).filter(Boolean);

  function onScroll(){
    const y = window.scrollY;
    const h = document.documentElement.scrollHeight - window.innerHeight;
    progressBar.style.width = (h>0 ? (y/h*100) : 0) + '%';
    if (y > 600) backTop.classList.add('visible'); else backTop.classList.remove('visible');
    const navH = nav.offsetHeight + 10;
    let active = sections[0];
    for (const s of sections){
      if (s && (s.getBoundingClientRect().top - navH) <= 0) active = s;
    }
    links.forEach(l => {
      l.classList.toggle('active', active && l.dataset.target === active.id);
    });
    const activeLink = links.find(l => l.classList.contains('active'));
    if (activeLink){
      const navInner = nav.querySelector('.topnav-inner');
      const linkRect = activeLink.getBoundingClientRect();
      const innerRect = navInner.getBoundingClientRect();
      if (linkRect.right > innerRect.right - 20 || linkRect.left < innerRect.left +
