  <div class="section-title aos"><span class="num">21</span><h2>📖 Hướng dẫn đọc — Giải thích thuật ngữ</h2><div class="divider"></div></div>
  <div class="aos">
    <div class="glossary-grid">
      <div class="gloss-item"><div class="gloss-term">P/E (Price-to-Earnings)</div><div class="gloss-def">Giá / EPS. Cho biết bạn trả bao nhiêu năm lợi nhuận để mua cổ phiếu. P/E 13x = 13 năm.</div></div>
      <div class="gloss-item"><div class="gloss-term">P/B (Price-to-Book)</div><div class="gloss-def">Giá / Vốn chủ sở hữu trên cổ phiếu. Đo lường giá trị tài sản ròng. P/B 3.2x = trả 3.2 lần giá trị sổ sách.</div></div>
      <div class="gloss-item"><div class="gloss-term">ROE (Return on Equity)</div><div class="gloss-def">Lợi nhuận / Vốn chủ sở hữu. Hiệu quả sinh lời trên vốn cổ đông. ROE 25% = mỗi 100đ vốn tạo 25đ lợi nhuận.</div></div>
      <div class="gloss-item"><div class="gloss-term">FCF (Free Cash Flow)</div><div class="gloss-def">CFO - Capex. Tiền mặt dư sau khi đầu tư. FCF dương = công ty có tiền trả cổ tức, mua lại cổ phiếu.</div></div>
      <div class="gloss-item"><div class="gloss-term">Capex (Capital Expenditure)</div><div class="gloss-def">Chi phí đầu tư tài sản cố định (xây nhà máy, mua thiết bị). Capex cao = đang mở rộng, nhưng giảm FCF ngắn hạn.</div></div>
      <div class="gloss-item"><div class="gloss-term">Midstream Gas</div><div class="gloss-def">Giai đoạn trung chuyển trong chuỗi dầu khí: vận chuyển, lưu kho, xử lý khí từ mỏ đến khách hàng.</div></div>
      <div class="gloss-item"><div class="gloss-term">LNG (Liquefied Natural Gas)</div><div class="gloss-def">Khí thiên nhiên hóa lỏng. Nhập khẩu từ nước ngoài, tái hóa lỏng tại cảng, sau đó phân phối qua đường ống.</div></div>
      <div class="gloss-item"><div class="gloss-term">Cơ chế giá K-định mức</div><div class="gloss-def">Công thức tính giá gas do Chính phủ quy định, bao gồm chi phí thu mua + biên lợi nhuận chuẩn cho midstream.</div></div>
      <div class="gloss-item"><div class="gloss-term">RSI (Relative Strength Index)</div><div class="gloss-def">Chỉ báo kỹ thuật đo sức mạnh tương đối. RSI >70 = quá mua, <30 = quá bán.</div></div>
      <div class="gloss-item"><div class="gloss-term">Drawdown</div><div class="gloss-def">Phần trăm giảm từ đỉnh giá. Drawdown 30% = từ đỉnh, giá giảm 30%.</div></div>

(function(){
  const nav = document.getElementById('topnav');
  const links = Array.from(document.querySelectorAll('.topnav-link'));
  const progressBar = document.getElementById('progressBar');
  const backTop = document.getElementById('backTop');
  const sections = links.map(l => document.getElementById(l.dataset.target)).filter(Boolean);

  function onScroll(){
    const y = window.scrollY;
    const h = document.documentElement.scrollHeight - window.innerHeight;
    progressBar.style.width = (h>0 ? (y/h*100) : 0) + '%';
    if (y > 600) backTop.classList.add('visible'); else backTop.classList.remove('visible');
    const navH = nav.offsetHeight + 10;
    let active = sections[0];
    for (const s of sections){
      if (s && (s.getBoundingClientRect().top - navH) <= 0) active = s;
    }
    links.forEach(l => {
      l.classList.toggle('active', active && l.dataset.target === active.id);
    });
    const activeLink = links.find(l => l.classList.contains('active'));
    if (activeLink){
      const navInner = nav.querySelector('.topnav-inner');
      const linkRect = activeLink.getBoundingClientRect();
      const innerRect = navInner.getBoundingClientRect();
      if (linkRect.right > innerRect.right - 20 || linkRect.left < innerRect.left + 20){
        navInner.scrollTo({ left: navInner.scrollLeft + (linkRect.left - innerRect.left - 40), behavior:'smooth' });
      }
    }
  }
  window.addEventListener('scroll', onScroll, {passive:true});
  window.addEventListener('resize', onScroll);
  links.forEach(l => {
    l.addEventListener('click', (e)=>{
      const t = document.getElementById(l.dataset.target);
      if (!t) return;
      e.preventDefault();
      const top = t.getBoundingClientRect().top + window.scrollY - nav.offsetHeight - 8;
      window.scrollTo({ top, behavior:'smooth' });
    });
  });
  backTop.addEventListener('click', ()=> window.scrollTo({top:0, behavior:'smooth'}));
  onScroll();
})();
</script>
