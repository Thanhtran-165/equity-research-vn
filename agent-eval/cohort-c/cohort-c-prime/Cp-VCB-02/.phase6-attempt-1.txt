<section id="sec-glossary" class="section">
  <div class="section-title aos"><span class="num">21</span><h2>📖 Hướng dẫn đọc — Giải thích thuật ngữ</h2><div class="divider"></div></div>
  <div class="section-sub">Thuật ngữ tài chính & ngân hàng dùng trong báo cáo.</div>
  <div class="glossary-grid aos">
    <div class="gloss-item"><div class="gloss-term">P/E (Price/Earnings)</div><div class="gloss-def">Giá / EPS. Cho biết bạn trả bao nhiêu cho 1 đồng lợi nhuận. P/E 18× = trả 18 đồng cho 1 đồng lợi nhuận.</div></div>
    <div class="gloss-item"><div class="gloss-term">P/B (Price/Book)</div><div class="gloss-def">Giá / BVPS. Cho biết bạn trả bao nhiêu so với giá trị sổ sách. P/B 1.0× = trả đúng giá trị sổ sách.</div></div>
    <div class="gloss-item"><div class="gloss-term">EPS</div><div class="gloss-def">Lợi nhuận trên mỗi cổ phiếu = Lợi nhuận / Số cổ phiếu.</div></div>
    <div class="gloss-item"><div class="gloss-term">ROE</div><div class="gloss-def">Lợi nhuận / Vốn chủ sở hữu. Đo lường hiệu quả sử dụng vốn. ROE 20% = 1 đồng vốn tạo 0.2 đồng lợi nhuận.</div></div>
    <div class="gloss-item"><div class="gloss-term">NIM</div><div class="gloss-def">Biên lãi thuần = (Thu nhập lãi - Chi phí lãi) / Tài sản sinh lãi TB. Đo lường spread huy động-cho vay.</div></div>
    <div class="gloss-item"><div class="gloss-term">NPL</div><div class="gloss-def">Tỷ lệ nợ xấu = Nợ xấu / Tổng dư nợ. NPL cao = rủi ro tín dụng cao.</div></div>
    <div class="gloss-item"><div class="gloss-term">CASA</div><div class="gloss-def">Tiền gửi không kỳ hạn / Tổng tiền gửi. CASA cao = nguồn vốn giá rẻ.</div></div>
    <div class="gloss-item"><div class="gloss-term">SOCB</div><div class="gloss-def">State-Owned Commercial Bank — Ngân hàng thương mại nhà nước. VCB, BID, CTG, Agribank.</div></div>
    <div class="gloss-item"><div class="gloss-term">JSCB</div><div class="gloss-def">Joint-Stock Commercial Bank — Ngân hàng thương mại cổ phần. TCB, MB, VPB, HDB...</div></div>
    <div class="gloss-item"><div class="gloss-term">Split-adjusted</div><div class="gloss-def">Dữ liệu giá/EPS đã được điều chỉnh cho cổ tức bằng cổ phiếu hoặc stock split. Cần kiểm tra khi so sánh lịch sử.</div></div>
  </div>

(function(){
  const nav = document.getElementById('topnav');
  const links = Array.from(document.querySelectorAll('.topnav-link'));
  const progressBar = document.getElementById('progressBar');
  const backTop = document.getElementById('backTop');
  const sections = links.map(l => document.getElementById(l.dataset.target)).filter(Boolean);

  function onScroll(){
    const y = window.scrollY;
    const h = document.documentElement.scrollHeight - window.innerHeight;
    progressBar.style.width = (h>0 ? (y/h*100) : 0) + '%';
    if (y > 600) backTop.classList.add('visible'); else backTop.classList.remove('visible');
    const navH = nav.offsetHeight + 10;
    let active = sections[0];
    for (const s of sections){
      if (s && (s.getBoundingClientRect().top - navH) <= 0) active = s;
    }
    links.forEach(l => {
      l.classList.toggle('active', active && l.dataset.target === active.id);
    });
    const activeLink = links.find(l => l.classList.contains('active'));
    if (activeLink){
      const navInner = nav.querySelector('.topnav-inner');
      const linkRect = activeLink.getBoundingClientRect();
      const innerRect = navInner.getBoundingClientRect();
      if (linkRect.right > innerRect.right - 20 || linkRect.left < innerRect.left + 20){
        navInner.scrollTo({ left: navInner.scrollLeft + (linkRect.left - innerRect.left - 40), behavior:'smooth' });
      }
    }
  }
  window.addEventListener('scroll', onScroll, {passive:true});
  window.addEventListener('resize', onScroll);
  links.forEach(l => {
    l.addEventListener('click', (e)=>{
      const t = document.getElementById(l.dataset.target);
      if (!t) return;
      e.preventDefault();
      const top = t.getBoundingClientRect().top + window.scrollY - nav.offsetHeight - 8;
      window.scrollTo({ top, behavior:'smooth' });
    });
  });
  backTop.addEventListener('click', ()=> window.scrollTo({top:0, behavior:'smooth'}));
  onScroll();
})();
</script>
