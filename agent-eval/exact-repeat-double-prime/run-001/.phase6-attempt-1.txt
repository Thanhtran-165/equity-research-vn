<section id="sec-glossary" class="section">
  <div class="section-title aos"><span class="num">21</span><h2>📖 Hướng dẫn đọc — Giải thích thuật ngữ</h2><div class="divider"></div></div>
  <div class="aos">
    <div class="glossary-grid">
      <div class="gloss-item"><div class="gloss-term">P/E (Price/Earnings)</div><div class="gloss-def">Giá / EPS. P/E 3.91× nghĩa nhà đầu tư trả 3.91 VND cho mỗi 1 VND lợi nhuận. Thấp = rẻ (nhưng có thể là bẫy chu kỳ).</div></div>
      <div class="gloss-item"><div class="gloss-term">P/B (Price/Book)</div><div class="gloss-def">Giá / Vốn CSH trên cổ phiếu. P/B 1.0× = mua đúng giá trị sổ sách. Dưới 1 = rẻ hơn giá trị sổ.</div></div>
      <div class="gloss-item"><div class="gloss-term">ROE (Return on Equity)</div><div class="gloss-def">Lợi nhuận / Vốn CSH. ROE 19% = mỗi 100 VND vốn tạo ra 19 VND lợi nhuận/năm. >15% là tốt.</div></div>
      <div class="gloss-item"><div class="gloss-term">EPS (Earnings Per Share)</div><div class="gloss-def">Lợi nhuận / Số cổ phiếu. EPS 18,357 VND = mỗi cổ phiếu tạo ra 18,357 VND lợi nhuận năm 2025.</div></div>
      <div class="gloss-item"><div class="gloss-term">Backlog</div><div class="gloss-def">Tổng giá trị hợp đồng đã ký nhưng chưa ghi nhận doanh thu. Backlog cao = đảm bảo doanh thu tương lai.</div></div>
      <div class="gloss-item"><div class="gloss-term">CFO (Cash Flow from Operations)</div><div class="gloss-def">Dòng tiền từ hoạt động kinh doanh. CFO âm = công ty chi nhiều hơn thu (phổ biến ở ngành xây dựng).</div></div>
      <div class="gloss-item"><div class="gloss-term">Capex (Capital Expenditure)</div><div class="gloss-def">Chi tiêu vốn (mua tài sản cố định, thiết bị). Capex thấp = asset-light, linh hoạt hơn.</div></div>
      <div class="gloss-item"><div class="gloss-term">Net Margin</div><div class="gloss-def">Lợi nhuận sau thuế / Doanh thu. Net margin 5.3% = mỗi 100 VND doanh thu tạo ra 5.3 VND lợi nhuận.</div></div>
      <div class="gloss-item"><div class="gloss-term">Drawdown</div><div class="gloss-def">Khoảng giảm từ đỉnh giá nhất xuống đáy. Drawdown -35% = cổ phiếu giảm 35% từ đỉnh.</div></div>
      <div class="gloss-item"><div class="gloss-term">Split-adjusted</div><div class="gloss-def">Giá đã điều chỉnh theo chia tách cổ phiếu. Cần dùng split-adjusted price để so sánh lịch sử chính xác.</div></div>
      <div class="gloss-item"><div class="gloss-term">Cross-check</div><div class="gloss-def">Kiểm tra chéo nhiều nguồn độc lập để xác minh số liệu. Tránh tin 1 nguồn duy nhất.</div></div>
      <div class="gloss-item"><div class="gloss-term">Bẫy 5F</div><div class="gloss-def">Fake (giả), Fragile (mong manh), Fickle (thay đổi), Financialized (tài chính hóa), Fleeting (ngắn hạn) — 5 dấu hiệu cảnh báo chất lượng kinh doanh kém.</div></div>

function grad(ctx, c1, c2){
  const chart = ctx.chart;
  const {ctx: cc, chartArea} = chart;
  if (!chartArea) return c1;
  const g = cc.createLinearGradient(0, chartArea.top, 0, chartArea.bottom);
  g.addColorStop(0, c1); g.addColorStop(1, c2);
  return g;
}
