## 2. Benchmark Philosophy

This is a **public engagement benchmark only**. We compare:
- ✅ Followers, posting frequency, reactions, comments, shares
- ✅ Public video views, engagement per post, engagement per follower
- ✅ Share ratio, comment ratio, viral hit rate

We **cannot** compare:
- ❌ True reach, true impressions, CTR, true watch time
- ❌ Negative feedback, organic/paid split, unique viewers

## 3. Candidate Scoring Methodology

Each candidate scored on 1–5 scale:

| Factor | Weight | Description |
|---|---|---|
| audienceOverlapScore | 30% | How close is audience to Chim Cút |
| contentSimilarityScore | 25% | Topic/format similarity |
| benchmarkUsefulnessScore | 25% | Value of comparison |
| dataAvailabilityScore | 15% | Public data accessibility |
| riskScore | -5% | Noise/risk penalty |

`totalFitScore` = weighted sum. Tier: A ≥ 3.3, B ≥ 2.8, C ≥ 2.3, D < 2.3.
